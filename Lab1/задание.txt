1. Создание клиентской части многостраничного сайта без
использования технологий транспиляции кода
Создать страницы для сайта, согласно своему варианту, используя
только HTML5, CCS3.
Тематический сайт имеет несколько Категорий. В каждой Категории
есть список Контента.
Тематический сайт состоит из:
• корневая домашняя страница index.html;
• страницы Категорий, имеющих список Контента из выбранной
Категории;
• страницы Контента;
Из страниц типа Категории и Контента достаточно реализовать по
одной рабочей.
Все страницы должны иметь одинаковый дизайн и иметь общий
подключаемый файл со стилями. Дизайн должен быть адаптивен под
различные разрешения экрана. Страницы должны состоять из блоков шапки,
основного наполнения и подвала. Должен присутствовать на страницах блок
с навигацией.

Вариант 10
Вы разрабатываете сайт магазина тематических товаров (мерчей).
В качестве Категорий выступают тематики представленных товаров.
В каждой Категории присутствуют товары в качестве основного Контента.
Каждый товар содержит название в качестве Заголовка, фотографию в
качестве Картинки, описание товара или некоторые тематические вставки в
качестве Текстового Описания и тип товара в качестве Свойства с
возможными значениями: Кружка, Футболка, Фигурка.
